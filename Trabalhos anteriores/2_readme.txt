Orientacao de compilacao e execucao

Abrir duas instancias do Terminal

Em uma delas, compilar o codigo usando a linha de comando: gcc head_or_tail.c -o head_or_tail

Para rodar o programa:
	-Em uma das instancias to terminal entre o comando: ./head_or_tail -S [ ]
		Isso roda a instancia do server do codigo;
	-Na outra instancia entre com o comando: ./head_or_tail host [ ]
		Isso roda a instancia do cliente do codigo;
